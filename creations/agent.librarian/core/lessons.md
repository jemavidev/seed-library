# Lessons — agent.librarian

No operational lessons yet (hand-seeded 2026-07-07). Adapted from the `ask-matt` router in
`mattpocock/skills` (MIT), with one deliberate change for a self-evolving library: the index is
**generated**, not hand-written, because a self-evolving library adds assets forever and a
hand-maintained router would rot by the next wave (see [[cognitive-vs-context-load]]). The curated flows are the only
hand-tended part, and they name journeys, not assets.
