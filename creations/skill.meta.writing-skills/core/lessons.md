# Lessons — writing-skills

No operational lessons yet (hand-seeded 2026-07-07). This creation is the home of the
authoring doctrine adapted from `mattpocock/skills` (MIT); the load framing, leading words,
and failure modes live as standalone library lessons
([[cognitive-vs-context-load]], [[leading-words]], [[skill-failure-modes]]) so any asset can
link them, and this skill assembles them into an authoring procedure.

A self-evolving system that fabricates skills forever needs a skill for fabricating skills —
that reflexivity is why this lives in the `meta` domain and why the generation engine should
consult it by default.
